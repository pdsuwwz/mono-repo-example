# mono-repo-example
The Architecture of Monorepo System based on Lerna


## Install

```bash
yarn install
```

## Run app

```bash
node .
```

## Tests with Lerna

```bash
yarn test
```

## Lint

```bash
yarn lint
```

## Update Version

```bash
yarn update:version
```

## Install External Dependencies

```bash
yarn add xxx -W
```
